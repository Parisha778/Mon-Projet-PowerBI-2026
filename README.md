# Mon Projet Power BI - Analyse des Ventes Europe

## Description
Ceci est mon premier tableau de bord interactif réalisé sur Power BI. 
Il contient :
- Une analyse du Chiffre d'Affaires (CA) par produit et par catégorie.
- Une répartition des ventes par pays.
- Des filtres dynamiques (Style Tuile) pour naviguer entre les catégories (Beauté, Électronique, etc.).

## Installation
Pour ouvrir ce projet sur un nouvel ordinateur :
1. Télécharger le fichier `.pbix`.
2. Télécharger le fichier `europe_sales_data.xlsx`.
3. Ouvrir le fichier `.pbix` avec Power BI Desktop.
4. Si les données ne s'affichent pas, aller dans **Paramètres de la source de données** et pointer vers le fichier Excel téléchargé.
